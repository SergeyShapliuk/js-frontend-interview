# JavaScript Frontend Interview Questions

## General Questions
1. What is the difference between `==` and `===`?
2. Explain the concept of closures in JavaScript.
3. What are the different data types in JavaScript?

## DOM Manipulation
1. How do you select an element in the DOM?
2. What is event delegation?
3. How can you prevent the default action of an event?

## Asynchronous JavaScript
1. What is the event loop?
2. Explain the difference between `setTimeout` and `setInterval`.
3. What are Promises and how do they work?

## ES6 Features
1. What are arrow functions and how do they differ from regular functions?
2. Explain the concept of template literals.
3. What are destructuring and spread operator?

## Frameworks and Libraries
1. What's the difference between React and Angular?
2. What are the advantages of using frameworks like React?
3. Can you explain the concept of Virtual DOM?

## Miscellaneous
1. What is the use of `this` keyword in JavaScript?
2. Explain how `localStorage` and `sessionStorage` differ.
3. What are service workers?
